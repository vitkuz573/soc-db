    **HTTPSource** class:

    ```python
    class HTTPSource:
        """Tiered HTTP fetcher: httpx → curl-cffi → Playwright.
        
        Each tier is tried in order. If a tier raises a retryable exception
        (ConnectionError, TimeoutError, HTTP 403/429), the next tier is tried.
        If all tiers fail, the last exception propagates.
        """
        
        def __init__(
            self,
            rate_limiter: PerSourceRateLimiter | None = None,
            default_headers: dict[str, str] | None = None,
            timeout: float = 30.0,
            follow_redirects: bool = True,
            cache_ttl: int = 86400,
        ):
            self._rate_limiter = rate_limiter
            self._timeout = timeout
            self._follow_redirects = follow_redirects
            self._cache_ttl = cache_ttl
            self._logger = logging.getLogger(__name__)
    ```

    Key design points:

    a) **Tier 1 — httpx (sync)**:
    - Use `httpx.Client` with `follow_redirects=True`, `timeout=timeout`, custom headers.
    - On HTTP 403 or 429: NOT retried with httpx (escalate to Tier 2).
    - On ConnectionError, TimeoutError, DNS failure: log and escalate.
    - On other HTTP errors (5xx): retry with tenacity via the rate limiter's retry_decorator, then escalate if still failing.
    - Close client after each request or use context manager.

    b) **Tier 2 — curl-cffi (sync requests mode)**:
    - Use `curl_cffi.requests.Session` with `impersonate="chrome120"` (closest to common browser TLS fingerprint).
    - Set the same headers as Tier 1.
    - On any failure: log and escalate to Tier 3.
    - Note: curl-cffi is an optional dependency. If `import curl_cffi` raises ImportError, skip Tier 2 silently and go to Tier 3.

    c) **Tier 3 — Playwright (sync)**:
    - Use `playwright.sync_api.sync_playwright()` context manager.
    - Launch `chromium.launch(headless=True)`.
    - Navigate to URL with `page.goto(url, wait_until="domcontentloaded")`.
    - Return `page.content()` (full HTML).
    - On any failure: log and raise the exception (last tier).
    - Note: Playwright is an optional dev dependency. If `import playwright` raises ImportError, raise a clear RuntimeError: "Playwright not installed. Run: pip install playwright && playwright install chromium".

    **Methods:**

    a) `fetch(self, url: str, user_agent: str | None = None) -> str`:
    - Acquire rate limiter if configured (call `rate_limiter.acquire()`).
    - Run the tier chain:
      - If Tier 1 succeeds: call `rate_limiter.record_success()`, return content.
      - If Tier 1 fails and escalation is needed: try Tier 2.
      - If Tier 2 fails (or unavailable): try Tier 3.
      - If all tiers fail: call `rate_limiter.record_failure()`, reraise last exception.
    - The outer call is wrapped in the rate limiter's `retry_decorator` to handle transient network errors.
    - After success, update `rate_limiter.record_success()`.

    b) `_fetch_httpx(self, url: str, user_agent: str) -> tuple[bool, str | None, Exception | None]`:
    - Returns `(success, content_or_None, exception_or_None)`.
    - Catches only retryable exceptions; raises immediately on non-retryable.

    c) `_fetch_curl(self, url: str, user_agent: str) -> tuple[bool, str | None, Exception | None]`:
    - Same return pattern.
    - Gracefully handles missing curl-cffi (ImportError → return (False, None, None), no log).

    d) `_fetch_playwright(self, url: str, user_agent: str) -> str`:
    - Returns content or raises.

    e) `_should_escalate(self, status_code: int) -> bool`:
    - Returns True for 403, 429, 503.
    - These indicate anti-bot blocking, so escalation to browser impersonation is appropriate.

    **Tier escalation policy (per per RESEARCH.md: "curl-cffi handles anti-bot at the HTTP layer without browser overhead"):**
    - httpx (fast, HTTP/2, standard) → try first
    - curl-cffi (Chrome TLS fingerprint, anti-bot) → try on 403/429 or network errors
    - Playwright (full browser) → try only as last resort

    **Cache integration:**
    - Before any HTTP call, check disk cache (same as common.fetch's CACHE_DIR, keyed by URL MD5).
    - Return cached content if within TTL and user_agent matches.
    - On fetch success, write to cache (respect guard_path from common.py).

    **User-Agent:**
    - Accept optional user_agent parameter per call. Falls back to common.USER_AGENT.
    - For curl-cffi: keep the same UA (the TLS impersonation handles fingerprint, the UA is additional identity).
    - For Playwright: use the same UA (Playwright's browser context shows a real browser UA).

    Import guard for curl-cffi:
    ```python

        import curl_cffi.requests  # noqa: F401
        _HAS_CURL = True
    except ImportError:
        _HAS_CURL = False
    ```

    Import guard for playwright:

    ```python
        _HAS_PLAYWRIGHT = True
    except ImportError:
        _HAS_PLAYWRIGHT = False
    ```

    ```python
    class SourceRegistry:
        """Registry of BaseScraper subclasses, auto-discovered by convention.
        
        Discovery convention: any module in sources/ that exports a class
        with a SOURCE_ID class attribute gets registered. The class must be
        a subclass of BaseScraper (enforced at registration time).
        """
    ```

    Methods:

    a) `register(cls, scraper_cls: type[BaseScraper]) -> None`:
    - Validates that `scraper_cls` is a subclass of `BaseScraper` (raises TypeError if not).
    - Validates that `scraper_cls.SOURCE_ID` is non-empty (raises ValueError if empty).
    - Stores in `cls._registry[scraper_cls.SOURCE_ID] = scraper_cls`.
    - Logs registration at INFO level.

    b) `discover(cls) -> dict[str, type[BaseScraper]]`:
    - Scans `soc_db.scraping.sources` module for all public classes that are BaseScraper subclasses.
    - Uses `importlib.import_module("soc_db.scraping.sources")` and `pkgutil.walk_packages` or manual inspection of the module.
    - For each discovered class, calls `cls.register()`.
    - Returns the registry dict (copy).
    - If the sources package doesn't exist yet (no sources/__init__.py), return empty dict — this is a valid state.

    c) `get(cls, source_id: str) -> type[BaseScraper] | None`:
    - Returns the scraper class or None if not found.

    d) `all(cls) -> list[type[BaseScraper]]`:
    - Returns all registered scraper classes sorted by SOURCE_ID.

    e) `clear(cls) -> None`:
    - Clears the registry (useful for testing).

    Internal: `_registry: ClassVar[dict[str, type[BaseScraper]]] = {}`.

    **src/soc_db/scraping/drift.py** — SchemaDriftDetector for alerting on page structure changes.

    ```python
    class SchemaDriftDetector:
        """Detects when a source's page structure changes by comparing
        actual fields found against expected fields.
        
        Each source registers the set of fields it expects to find.
        After each scrape run, the detector compares actual vs expected.
        If the intersection is < threshold (default 80%), a warning is logged.
        """
    ```

    Methods:

    a) `register_expected(self, source_id: str, fields: set[str]) -> None`:
    - Stores `self._expected[source_id] = fields`.
    - If source_id already has registered fields and the new set differs by >10% from old, log INFO with the diff.

    b) `check(self, source_id: str, results: list[ChipScrapeResult]) -> DriftReport`:
    - If source_id not registered: log WARNING "No expected fields registered for {source_id}" and return DriftReport with `drift_detected=False`.
    - If results empty: log WARNING "Zero results for {source_id} — possible drift" and return DriftReport with `drift_detected=True`.
    - Otherwise:
      - Collect all field names from all results (union of `result.fields.keys()`).
      - Compute `expected = self._expected[source_id]`.
      - Compute `intersection = expected & actual_fields`.
      - Compute `coverage = len(intersection) / len(expected)`.
      - If coverage < self._threshold: log WARNING with details (source_id, coverage%, missing fields).
      - Return DriftReport.

    c) `DriftReport` dataclass (can be defined in the same file):
    ```python
    @dataclass
    class DriftReport:
        source_id: str
        drift_detected: bool
        coverage: float = 1.0
        expected_fields: set[str] = field(default_factory=set)
        actual_fields: set[str] = field(default_factory=set)
        missing_fields: set[str] = field(default_factory=set)
        new_fields: set[str] = field(default_factory=set)
        message: str = ""
    ```

    d) `__init__(self, threshold: float = 0.8)`:
    - `self._threshold = threshold` (clamped to [0.0, 1.0]).
    - `self._expected: dict[str, set[str]] = {}`.

    e) `expected_fields(self, source_id: str) -> set[str]`:
    - Returns registered fields or empty set if not registered.

    f) `clear(self) -> None`:
    - Clears all registered expected fields (for testing).

    Both files: use `from __future__ import annotations`. Use `logging.getLogger(__name__)`. Type hints throughout.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HTTPSource → External websites | Untrusted HTML crosses into scrapers; malicious payload could exploit parser |
| SourceRegistry → Import system | Auto-discovery uses importlib; a malicious sources/ module could execute code at import time |
| SchemaDriftDetector → Logging | Drift detection logs may contain scraped content; log injection risk |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-08-04 | Tampering | HTTPSource._fetch_playwright | medium | accept | Playwright is a headless browser; HTML with JS redirects won't harm the process. No eval of page content. |
| T-08-05 | DoS | HTTPSource tier chain | low | accept | If all tiers fail and timeouts cascade, a single URL could block the scraper for minutes. Mitigated by tenacity timeouts on each tier (30s default). |
| T-08-06 | Tampering | SourceRegistry.discover importlib | low | accept | sources/ is in the soc_db package — trusted code only. Not a user-writable directory. |
| T-08-07 | Information Disclosure | SchemaDriftDetector logs | low | accept | Logs contain field names only, not values. Field names are static (cpu, gpu, process_nm, etc.). |
| T-08-SC | Tampering | pip install playwright (dev dep) | medium | mitigate | Playwright is optional dev dependency. HTTPSource handles missing import gracefully with clear error message. |
</threat_model>
